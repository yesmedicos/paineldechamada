Como funciona

Do contrato ao painel na TV
em 3 passos simples

Você vende. Nós entregamos a tecnologia. Sem complicação técnica para o seu cliente.

01

Você fecha com seu cliente

Contrate o painel com sua marca e apresente para o seu cliente como uma solução própria. A plataforma e o conteúdo ficam no nome que você escolher.

02

Enviamos o aparelho

O dispositivo chega pronto para o cliente final. Basta conectar à TV via HDMI e à rede Wi-Fi. Sem técnico, sem instalação complicada.

03

Você configura online

Acesse a plataforma e personalize logomarca, cores, voz, vídeos e conteúdo. Gerencie múltiplos clientes, salas e painéis de um único lugar.

Integração via API

Conecte seu sistema
ao painel em minutos

Nossa API REST foi projetada para ser simples e direta. Uma única chamada HTTP já aciona a chamada no painel — sem SDK, sem instalação, sem complicação.

Uma chamada, resultado imediato

Envie um POST com o nome do paciente, número da senha ou qualquer dado — o painel exibe e anuncia por voz em tempo real.

Funciona com qualquer linguagem

Python, Node.js, PHP, Java, C# — qualquer sistema que faça requisições HTTP consegue integrar em minutos.

Prontuários e agendas médicas

Integre com o sistema de agenda da clínica e dispare chamadas automáticas no momento certo, sem intervenção manual.

Sistemas de senhas e filas

Bancos, cartórios e órgãos públicos podem conectar seu emissor de senhas e acionar o painel automaticamente a cada chamada.

Autenticação segura por API Key

Cada cliente recebe sua própria chave de API. Controle de acesso granular por painel, sala e cliente dentro da plataforma.

REST API — exemplo
# Acionar chamada no painel
POST https://api.paineldechamada.com.br/v1/call

# Headers
Authorization: Bearer {sua_api_key}
Content-Type: application/json

# Body (clínica)
{
  "panel_id": "painel-recepcao-01",
  "name": "Marcos Oliveira",
  "room": "Consultório 3",
  "voice": true
}

# Body (banco)
{
  "panel_id": "painel-agencia-sp",
  "ticket": "A-047",
  "desk": "Caixa 1",
  "voice": true
}

# Resposta
{
  "status": "ok",
  "called_at": "2025-09-12T10:47:03Z"
}
FAQ

Perguntas frequentes

Preciso ter conhecimento técnico para usar?

Não. A plataforma é 100% visual e intuitiva. Se você sabe usar um painel de controle simples, você consegue configurar e gerenciar todos os painéis.

Posso usar meu próprio nome de marca?

Sim. O produto é 100% white label. Você apresenta com o nome, logo e identidade que quiser. Seu cliente nem precisa saber que a tecnologia é nossa.

Como meu cliente final instala o aparelho?

É plug & play. O dispositivo chega pronto. Basta conectar ao cabo HDMI da TV e à rede Wi-Fi. Em menos de 5 minutos o painel está funcionando.

Posso gerenciar vários clientes ao mesmo tempo?

Sim. A plataforma foi feita para agências. Você visualiza e gerencia todos os clientes, salas e painéis em uma única tela, sem limite de clientes.

Funciona em qualquer TV?

Sim. Funciona em qualquer televisor ou monitor com entrada HDMI, independentemente da marca ou modelo.

Consigo alterar o conteúdo remotamente?

Sim. Toda alteração feita na plataforma é refletida em tempo real no painel do cliente, sem necessidade de acesso físico ao aparelho.

A API funciona com qualquer sistema?

Sim. Nossa API REST é compatível com qualquer linguagem ou plataforma que faça requisições HTTP — prontuários médicos, sistemas de senha, ERPs, CRMs e muito mais.
